Usage

JustJs is available on npm for server usage and distributed over a CDN for browser usage.

Browser

To use the latest version of JustJs' core over a CDN, include the following anywhere on your HTML:

<script
src="https://cdn.jsdelivr.net/npm/@just-js/just@1.1.0-rc.1/dist/browser/core.min.js"
integrity="sha384-P7HOEj4Wg8jQlAlNw0RzuCdznnW3zREDidjDdaYJBgu75xPIyNC+lVI9lSoi3vJB"
crossorigin="anonymous"
async></script>

Then, you can replace core.min.js with any of these files and/or 1.1.0-rc.1 with any of these versions.

Versioning

JustJs follows the Semantic Versioning Specification (SemVer) to version each release. However, we tag the release prepending "v" to the version.

Usually, you should avoid using release-candidates (those releases that include "-rc" on the version) and use a previous/next version instead, since those are used for testing purposes and removed after a release is no longer a candidate.

If you need more information about a certain version, please check the source code on Github.
